A MacBook-style web desktop where apps run as draggable and resizable windows.

Try it live :)
https://appbook-bice.vercel.app/

---

## Features

- MacBook-style desktop UI  
- Draggable & resizable windows  
- Dynamic dock with app icons  
- Modular apps system for easy addition  
- Fast, responsive, minimal design
